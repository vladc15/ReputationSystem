# ReputationSystem

## Hardhat configuration

First, run:
```
npm install
```
To run the node:
```
npx hardhat node
```

To deploy the contracts run: 
```
npx hardhat run scripts/Deploy.js --network localhost
```

To start the frontend, run:
```
npm start
```
